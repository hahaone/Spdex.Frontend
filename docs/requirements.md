# Spdex Frontend - 需求文档

## 项目概述

Spdex（Sports Data Exchange）是一个体育数据展示与分析平台的前端项目。

## 业务需求

### 核心功能

1. **体育数据展示** - 展示来自 Spdex.WebApi 后端的体育数据
2. **数据分析** - 提供数据可视化和分析功能
3. **用户交互** - 响应式界面，支持多端访问

### 非功能需求

- **性能**: 支持 SSR（服务端渲染），首屏加载快速
- **兼容性**: 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- **响应式**: 适配桌面端和移动端
- **可维护性**: TypeScript 严格模式，统一代码规范（ESLint）

## 数据来源

- 后端 API: Spdex.WebApi（.NET），默认地址 `http://localhost:5000`
- 前端通过 `/api/**` 路由代理转发请求到后端
